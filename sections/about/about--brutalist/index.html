We ship raw code
for people who hate
waiting on a build.

Blockyard started as two engineers frustrated that every "starter kit" came with 400 dependencies and a config file longer than the actual app. So we deleted all of it. What's left is what you drop straight into Claude Code: no bundler, no framework tax, no npm install at 2am.

If it needs a build step to render a paragraph, it's already broken.
