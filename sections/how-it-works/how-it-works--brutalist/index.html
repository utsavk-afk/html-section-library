    [ COPY ]

    Grab the block

    Hit copy on any section. One self-contained HTML file lands on your clipboard — styles baked in, nothing to install.

  2. 02
    [ DROP ]

    Paste into Claude

    Drop it straight into Claude Code and say what to change. It reads the tokens and edits in place.

  3. 03
    [ TWEAK ]

    Swap the tokens

    Recolor, rewrite the copy, retune spacing. Every knob lives in one :root block up top — no hunting.

  4. 04
    [ SHIP ]

    Push it live

    Open the file in a browser, drag it to any host. No bundler, no npm, no framework tax.
