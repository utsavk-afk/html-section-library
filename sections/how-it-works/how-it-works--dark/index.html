The workflow

Paste it in.
Ship it the same day.

Every section in the library is a single self-contained file. No install, no config, no dependency graph to babysit — just four honest steps from copy to production.

~4 min copy to preview
0 build steps
  1. 01

    Grab the section

    Copy

    Open the file, select all, copy. One index.html holds the markup, tokens, and styles — nothing else to fetch.

  2. 02

    Drop it into Claude Code

    Paste

    Hand it to the agent with a one-line ask — "make the headline about our onboarding flow." It rewrites the copy and keeps the structure intact.

  3. 03

    Retint the tokens

    Adjust

    Every color lives in one :root block. Swap --accent, nudge the radius, match your brand in a couple of edits.

  4. 04

    Ship to production

    Deploy

    It's plain HTML and CSS, so it runs anywhere a browser does — paste into your page, commit, push. Done before the coffee's cold.
