How we got here

Five releases that shaped the way the toolkit works today.

Starting as a single formatting script in 2021, the library grew one deliberate step at a time. No rewrites, no pivots — just steady additions people actually asked for.

2021March

The first snippet ships

A 40-line prompt for cleaning up meeting notes, pasted into a public gist. It got 300 stars in a weekend and made the case for a proper collection.

2022January

Twelve prompts become a folder

The scattered snippets moved into a versioned repo with a README and copy buttons. Contributions opened up, and the first ten pull requests landed within a month.

2023September

Categories and search arrive

With 180 entries, browsing broke down. We added tags, a fuzzy search box, and a rule that every prompt ships with a real before-and-after example.

2024June

Drop-in for Claude Code

Each section became a self-contained file you could paste straight into your editor — no build step, no dependencies. Weekly installs crossed four thousand.

2026Today

Community-owned and freeNow

Governance handed to a working group of forty maintainers. Everything stays MIT-licensed, and the roadmap is decided in the open every quarter.
