Before you install

Questions from teams already running this in Claude Code.

Everything here is drawn from real support threads. If you hit something not covered, we usually reply the same working day.

Still stuck? Email hello@example.com

01 Do I need a build step to use the library?

No. Every section is a single self-contained index.html file. Copy it, paste it, open it in a browser — nothing to compile, no npm install, no bundler. The only external dependency is Google Fonts, loaded over a standard link tag.

02 Can I use these sections in commercial client work?

Yes. The library is free for commercial and personal projects, including paid client sites. You don't need to credit us, though a link back is always appreciated. You may not resell the collection itself as a competing template pack.

03 How do I change the accent colour and fonts?

Open the :root block at the top of the stylesheet. Every colour and typeface routes through a custom property — swap --accent to rebrand the whole section in one edit, or point --sans at a different family after updating the font link in the head.

04 Will this work with Tailwind or my existing CSS?

The markup uses plain, prefixed class names (faq__item, faq__q) so it won't collide with utility classes. Drop the section into a Tailwind or vanilla project and the scoped styles stay contained. If you'd rather go utility-first, the class structure maps cleanly one-to-one.

05 Is the accordion accessible and keyboard-friendly?

It's built on native <details> and <summary> elements, so expand and collapse work with the keyboard, screen readers announce state, and it degrades gracefully with JavaScript disabled. No custom ARIA wiring required — the browser handles focus and toggling for you.

06 How often do you add new sections?

New sections ship most weeks, usually in small batches grouped by type — heroes, pricing, testimonials, footers. Existing files get quiet updates when a browser default shifts. Watch the repo or check back; nothing you've already pasted will break underneath you.
