LAST UPDATE22.07.26
01 Is this library actually free to use?
Yes. Every section ships under an MIT license — use it in client work, resell the sites you build, strip the credit comment, whatever. No attribution required, no per-seat fee, no "pro tier" gate hiding half the components.
02 Do I need Tailwind, React, or a build step?
None of it. Each file is one self-contained index.html with inline CSS. Open it in a browser and it works. The only external request is a Google Fonts <link> — swap it for a local font and you're fully offline.
03 How do I drop a section into Claude Code?
Paste the raw HTML into your prompt and say "adapt this section to my brand tokens." Because every file uses a :root{} variable block — --bg, --ink, --accent — the model rewrites six values and the whole thing reskins in one shot.
04 Can I recolor it without breaking the layout?
That's the point of the token block. Change --accent from acid-yellow to hot pink and every marker, hover state, and highlight follows. Structure, spacing, and the hard offset shadows stay locked — you only ever touch color and font variables.
05 Is the markup accessible and keyboard-friendly?
This accordion is built on native <details> / <summary> — so it's tab-focusable, toggles on Enter/Space, and works with zero JavaScript. Screen readers announce expand/collapse state for free because the browser handles it.
06 What if a section breaks on my stack?
It shouldn't — there's no runtime to break. If something looks off, it's almost always a parent * { } reset colliding with the section. Wrap it in an isolating container or file an issue on the repo and it gets patched.
