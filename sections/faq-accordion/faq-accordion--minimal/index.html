01 Do I need a build step to use these sections?

No. Every section is a single self-contained HTML file — inline CSS, no bundler, no npm install. Open it in a browser or paste it straight into your project and it renders exactly as shown.

The only external dependency is Google Fonts, which you can swap for a local font in one line if you prefer to stay fully offline.

02 Can I use the sections in commercial client work?

Yes. The library is free for personal and commercial projects, including paid client sites. You don't need to credit us, though a mention is always appreciated.

The one thing you can't do is repackage the sections and resell them as a competing component library.

03 How do I change the accent color and fonts?

Every value lives in the :root block at the top of the file. Change --accent to recolor buttons, links, and markers in one edit, and swap the --font token to restyle all type.

Because the tokens cascade, you never have to hunt through the markup to update a color.

04 Are the sections responsive and accessible?

Each section ships with a mobile breakpoint at 640px and uses semantic HTML — this accordion is built on native <details> and <summary>, so keyboard and screen-reader support work out of the box.

Nothing overflows horizontally, and focus states are preserved rather than stripped.

05 Will these work inside React, Vue, or a CMS?

Yes. Since the markup is plain HTML and CSS, you can drop a section into a JSX component, a Vue template, or a CMS rich-text block. In React, just rename class to className and move the styles into your stylesheet.

06 How often do you add new sections?

We publish two to four new sections most weeks — heroes, pricing tables, testimonials, footers, and more — each in the same minimal, editorial style so they compose cleanly on one page.

Follow the changelog to see what shipped recently.

07 Can I request a section that isn't in the library yet?

Absolutely. Requests shape the roadmap more than anything else. Send a short description of the layout you need and a reference or two, and we'll queue it — most requested patterns land within a couple of weeks.
